?string : search of string in text

find

find command is used to search files and directories

$ find [path] -type fd -name “pattern”

$ find /home/user -type f -name "list*": searching in normal user, starting inside user, for a file that starts with file and ends with anything

$ find ./ -type f -name "list*": starting inside the current directory, for a file that starts with file and ends with anything

$ find ./ -type f -iname "list*": same thing but in either lowercase or uppercase

$ find ./September ./October -type f -iname "list*": specify as many places to search

$ find ]$ find -type f -name "list*": If no path is given, find looks in the current directory and its subdirectories

To suppress the permission denied messages, redirect standard error to 2>/dev/null

~]$ find /etc -type f -name "passwd" 2> /dev/null
